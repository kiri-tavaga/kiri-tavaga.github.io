JS Language

Parentheses

() Used to enclose an argument or parameter within a function or method.

Brackets

[] Used to index into an array.

Braces

{} Used to group statements and declarations. The contents of a class or interface are enclosed in braces. Used to group the statements in an IF statement, a LOOP, or other CONTROL STRUCTURE.

Single quotes

'' Create strings

Double quotes

"" Same as single quotes
