Scope

What is Scope?

Javascript has two scopes, global and local.

Scope refers to the accessibility of variables declared in Javascript.

Every function has it's own territory. If javascript was the planet Earth, a function would be a country.

Variables declared outside of functions are known as global scopes, which can be used anywhere in the Javascript code.

Variables declared inside functions are known as local scopes, meaning they are local to the function they are declared in and can only be used inside of that function.

Variables declared inside of the country/function can be accessed only by that country/function.
