What is a responsive site and why is responsiveness important?

In building the style and design of my new webpage I learnt about the importance of the page being responsive to any device being used.

A responsive site adapts in it's size to fit onto the screen of any device so that it is more user-friendly.

What is mobile first design and why is it important?

Mobile first design is beginning with the smallest screen size and device function.

Once the mobile design is created it will just need a few changes added on working towards the desktop size and functions.

It's important to design for mobile users first as it is the most common way of accessing the internet.

A mobile device functions differently to a desktop and it's layout will need to follow a different pattern with most important information first.

What are frameworks and their pros and cons?

A framework is a basic structure to build upon in creating a web page or application.

Pros.

  • It's reusable
  • Makes the process of building a site easier
  • It saves time

Cons:

  • It can add more time for your page to load
  • New features can be added by the owner of the framework that may affect your site.
